AI Notes

A full-stack AI note-taking application built with modern Angular architecture. Users can generate AI responses, save notes, view history, and navigate summaries with a clean, ChatGPT-style interface. The app emphasizes state-driven UI using Angular Signals, global loading management, and persistent storage.

The frontend is deployed as a single-page application on Netlify, while a secure Node.js backend proxy (hosted on Render) handles OpenAI requests using environment variables to prevent key exposure.

Angular • TypeScript • Signals • RxJS • Node.js • Express • OpenAI API • Netlify • Render

Bullet AI

An AI-powered resume bullet improver built with modern Next.js architecture. Users can enter an existing resume bullet and a job description line, and the application rewrites the bullet to better align with the role. The interface focuses on simplicity and fast iteration with tone options (Neutral, Technical, Impact) and persistent attempt history.

The application uses a Next.js API route as a secure backend layer that calls the Claude AI API. This keeps the API key protected on the server while allowing the frontend to request improvements through a simple /api/improve endpoint. Generated results are automatically saved in localStorage so users can review and compare previous attempts.

Next.js • React • TypeScript • Claude API • Serverless API Routes • Vercel • localStorage
